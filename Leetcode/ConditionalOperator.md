1. 319 Bulb Switcher

- https://leetcode.com/problems/bulb-switcher/description/

```js
var bulbSwitch = function(n) {
    // return parseInt(Math.sqrt(n));

    let count = 0;
    let i = 1;
    while((i*i) <= n){
        count++;
        i++;
    }
    return count;
};
```
